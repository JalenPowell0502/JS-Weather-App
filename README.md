# JS-Weather-App
<h1>Jalen Powell Weather Application</h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
Project consists of a simple JavaScript application that walks the user through retrieving weather information for any city. The utility allows you to input the city name and choose whether to display temperature, weather conditions, or humidity. The JavaScript application will configure an API call to the OpenWeatherMap service based on the user's input and then display the requested weather data (temperature, conditions, humidity) on the screen.
<br />


<h2>Languages and Utilities Used</h2>

- <b>JavaScript</b> 
- <b>OpenWeatherAPP API</b>

<h2>Environments Used </h2>

- <b>VS Code</b> (21H2)
